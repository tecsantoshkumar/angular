About Networking Course

The computer network is a system that connects many independent computers to exchange information (data) or resources. Computers and other devices can be integrated to make communication easier. The computer network is a collection of systems of computers that are connected together. You can establish a network connection using either wireless or cable media. Software and hardware are used to connect computers and other tools within a network.

There are many types of nodes in computer network. The computer network can include servers, networking hardware, personal computers, and other general-purpose hosts. They can be identified using their hostnames or network addresses.

There are two types of links

  • Physical (e.g. optical fibers or cable wires)
  • Virtual (free space used by wireless networks)
  • These physical and logical components are called computer network architecture. It describes the physical components of the network, its functional organization, and the procedures

The Goal of Computer Networking

  • Programs do not have to be executed on one system because of load and resource sharing
  • Lower costs: Multiple machines can share tape drives and printers
  • Reliability: If one machine is damaged, another machine can take its place
  • Scalability (it is easy to add more processors and computers)
  • People living apart can still communicate and work together via mail
  • Information Access (remote access to information, access the internet via e-mail and video conferencing
  • Interactive entertainment (online games, videos, etc.)
  • Social networking
