Core Java:

Core Java is the basic & raw part of the Java programming language that is used for creating or developing a general-purpose application. Using Core Java anyone can develop any advanced java application. The main topics covered in Core java are OOPS, data types, variables, operators, functions, classes & methods, loops, exception handling, threading, etc.

Advanced Java:

Advanced Java is also a part of the Java programming language that generally deals with online applications like websites and mobile applications. It deals with some specialization like Database, DOM(web), networking, etc. The main topics covered in advanced java like Servlet, JSP, JDBC, RMI, Socket programming, etc.

Web Technologies

Web technologies is the process of creating and building websites and web pages by combining any number of visual design elements including text, photos, graphics, animations, and videos. The three core technical languages that make up the World Wide Web are HTML, CSS, and JavaScript. The main topics covered in Web technologies like Fundamental Web Concepts, Client-side Programming, Server-side Programming, Web Data Communication and Processing.

Spring Framework

Spring is not a language. It is a framework of the Java programming language. Spring framework is widely used to develop software applications. It has a lot of components as well as it provides different functionalities. The Spring Components are Spring Model View Controller (MVC), Spring Core, Spring Transaction Management, Data Access Framework.
