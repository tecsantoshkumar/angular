What is MS-Office?

Microsoft Office is a collection of applications designed to help with productivity and completing common tasks on a computer. Each and every application of MS Office serves a unique purpose and offers a specific service to the business.

Microsoft Word – to create documents, flyers, publications.

Microsoft PowerPoint – to create & organize Rich looking Slides data including formulas, graphing tools, and more.

Microsoft Excel – Using & Creating Cell styles & Table styles.

OneDrive – to store data online.

Microsoft OneNote – to organize data collected including handwritten notes, drawings, screen captures, audio clips, and more.

Microsoft Outlook - to Create email and calendars, to-do lists, and contacts.

Microsoft Access – Create Database, & tables to compile and organize large amounts of data.

From the above list you can create so many combinations of apps that can be used together. For Example, you can write a document in Word and save it in the cloud using OneDrive. You can create an email in outlook and attach a file you’ve created in PowerPoint. Import contacts from Outlook to Microsoft Excel to create a spreadsheet of the people you know, their names, addresses, and so on.
