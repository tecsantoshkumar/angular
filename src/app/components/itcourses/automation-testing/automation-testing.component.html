What is Automation Testing?

Automation Testing is software testing that uses special automated software tools to run a test suite. Manual Testing, on the other hand, is done by a human who sits in front of a computer and carefully executes the test steps.

Automation testing software also allows you to enter test data into the System Under Test. This will allow you to compare actual and expected results, and generate detailed reports. Software Test Automation requires significant investments in money and resources.
